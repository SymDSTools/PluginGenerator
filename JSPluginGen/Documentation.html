MyJSPlugin

In order to use MyJSPlugin, you must first load the plugin at the top of your script using the LoadPlugin method like this:

 app.LoadPlugin( "MyJSPlugin" );

Then you can create an instance of the plugin object when you need it like this:

 plg = app.CreateMyJSPlugin();

Examples:

Example - Get Version

app.LoadPlugin( "MyJSPlugin" );

function OnStart()
{
    lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

    btn = app.CreateButton( "Get Plugin Version" );
    btn.SetOnTouch( CallPlugin );
    lay.AddChild( btn );

    plg = app.CreateMyJSPlugin();

    app.AddLayout( lay );
}

function CallPlugin()
{
    alert( plg.GetVersion() );
}
